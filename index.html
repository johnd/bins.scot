Fife Bins

This app lets you easily see which bins are being collected and when so you know what bin to put out. It gets this data by asking the Fife Council website. The app does not store any bin collection data locally, but it does store a special reference called a 'UPRN' which it uses to get the bin data from the Fife Council website.

This app is not endorsed or supported by Fife Council. If you have an issue with the app please contact the developer.
